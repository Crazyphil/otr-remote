Aufnahme ... abgehakt

Vor dem endgültigen Hinzufügen oder Löschen einer Aufnahme haben Sie - wenn Sie diese Einstellungen in den Programmoptionen aktiviert haben - noch einmal die Möglichkeit, sich alle Eigenschaften der Aufnahme anzeigen zu lassen und sie zu bearbeiten. Die Aufnahmevorschau ist auch der richtige Platz für Sie, Serienaufnahmen Ihrer Lieblingsserie zu starten.

Serienaufnahmen

Eine ganz besondere Spezialität bietet der Aufnahmevorschau-Dialog für Serienjunkies: Hier können Sie auf einfachste Weise eine ganze Staffel einer Serie inklusive der richtigen Episoden-Bezeichnungen aufnehmen!

Um die Serienaufnahmefunktion zu aktivieren, geben Sie die Anzahl der auf die aktuell eingetragene Aufnahme folgenden Tage ein, die in die Berechnung der Folgen mit einbezogen werden sollen. Wählen Sie außerdem aus, in welchem Intervall diese Folgeaufnahmen getätigt werden sollen. Die Intervalle sind:

 Für die Folgeaufnahmen werden die Einstellungen der aktuellen Aufnahme 1:1 kopiert, nur das Datum wird dabei jeweils angepasst. Anschließend werden alle Aufnahmen auf ein Mal aufgenommen, das heißt, Sie müssen OTR Remote nur für eine einzige Aufnahme starten, konfigurieren die Serienaufnahme und können eine ganze Staffel in einem Rutsch hinzufügen!

Wenn Sie die genaue Anzahl der Tage nicht ausrechnen wollen, klicken Sie auf den Pfeil nach oben im Eingabefeld für die Tage und bleiben Sie so lange darauf, bis unten links die richtige Aufnahmeanzahl angezeigt wird.

Organisieren der Serienaufnahmen

Wenn Sie Ihre Serienaufnahmen organisieren wollen, ist eines der häufigsten Sortiermuster Staffel- und Episodennummer. Der Aufnahmetitel (und in weiterer Folge auch der Dateiname) wäre so etwas wie Die Simpsons S05E21. Um alle Serienaufnahmen, die OTR Remote hinzufügt, in so ein Schema zu formen, muss es Ihnen möglich sein, eine Zahl im Titel für jede Aufnahme erhöhen zu können. OTR Remote ist dazu fähig, Zahlen im Aufnahmetitel (und nur im Titel) erhöhen zu können, wenn Sie spezielle Markierungen hinzufügen. Das sind die geschweiften Klammern { und }.

Auf das Beispiel von oben angewandt, würden Sie den Aufnahmetitel in OTR Remote hinzufügen, indem Sie Die Simpsons  S05E{21} schreiben. Das bedeutet, dass die erste Aufnahme die Nummer 21 hat, und jede darauf folgende Aufnahme, die in diesem Schritt hinzugefügt wird, eine um eins höhere Nummer bekommt. Natürlich können Sie so viele automatisch erhöhende Zahlen hinzufügen, wie Sie wollen. Dabei wird auch die Anzahl der Stellen berücksichtigt: Geben Sie {0001} ein, werden alle Aufnahmen eine vierstellige Nummer erhalten.
