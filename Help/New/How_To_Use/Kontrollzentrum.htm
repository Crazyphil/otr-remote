Das OTR Remote-Kontrollzentrum ist der zentrale Platz, an dem Sie alle Einstellungen konfigurieren, von Ihren OTR-Anmeldedaten bis zu den Netzwerkeinstellungen.

Das Wichtigste, das Sie konfigurieren sollten, sind Ihr OTR-Benutzername und -Passwort. Das Passwort wird DES-verschlüsselt auf Ihrer Festplatte gespeichert. Wenn Sie sie hier nicht eingeben, werden Sie Ihre Daten jedes Mal beim Hinzufügen einer Aufnahme über OTR Remote eingeben müssen.

Die nächste Einstellung ist die Zeitzone. Jedes mal, wenn Sie eine Aufnahme hinzufügen, müssen Start- und Endzeit angegeben werden. OTR kann die richtigen Zeiten automatisch unter Zuhilfename der Zeitzone berechnen, die Sie hier eingeben. Wenn Sie eine falsche Zeitzone eingeben, kann es sein, dass aufgrund der falsche Zeiten eine andere TV-Sendung als erwartet aufgenommen wird.

OTR Remote erlaubt es Ihnen, Plugins zu verwenden, damit Ihr EPG -Programm OTR Remote zum Aufnehmen von Sendungen verwendet. Die Auswahlliste zeigt alle aktuell verfügbaren Plugins.

Wenn Sie ein Plugin auswählen, ändert sich der "Plugineinstellungen"-Abschnitt. Dieser Abschnitt kann vom Plugin frei konfiguriert werden, daher müssen Sie sich bei Fragen über diese Einstellungen an die Hilfe des Plugins wenden.

Die "Optionen"-Schaltfläche öffnet den Programmoptionen-Dialog, in dem Sie Programmverhalten und Netzwerkeinstellungen festlegen und das Programm aktualisieren können. Die "Hilfe"-Schaltfläche öffnet dieses Dokument.

Wenn Sie die "OK"-Schaltfläche klicken, werden die Einstellungen gespeichert und das Plugin angewiesen, seine Arbeit zu tun. Wenn alles klappt, wird das Programm geschlossen. Wenn Sie die "Abbrechen"-Schalftläche klicken, werden jegliche Einstellungen (ausgenommen der im Optionen-Dialog) verworfen.

Wenn Sie die Auto-Aktualisieren-Einstellung aktiviert haben, wird Crazysoft AppUpdate jede Woche nach dem Schließen des Programms gestartet, um sicherzustellen, dass Ihre Version die aktuellste ist.
