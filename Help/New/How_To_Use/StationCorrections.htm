"And oh! I am glad that nobody knew..."

Every TV station has its own unique name. Has it? No, because nearly every EPG provider has own names for TV stations. May it be little changes (e.g. no dot in the station's name) or complete renames.

OTR, however, relies on the names it has for recordable TV stations to be able to identify TV shows correctly. OTR Remote has an internal database to compare the station name given by your EPG program and the one needed by OTR. For your comfort, this name is changed automatically when sending a request to OTR.

Using the Station Corrections dialog, you can edit this database to match your EPG's preferences.

Case-sensitive station name replacements: When checked, OTR Remote respects upper- and lowercase characters in the EPG station name when matching with the database.

By default, the station correction database is filled with the EPG station names of TVgenial and TV-Browser. If you add your own programs or corrections, we are glad if you send us the file stations.xml from OTR Remote's installation directory.
