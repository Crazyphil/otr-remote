Überwältigende Möglichkeiten

Der Aufnahmehilfe-Dialog wird angezeigt, wenn Sie OTR Remote mit dem Kommandozeilen-Parameter -h starten. Sie können nachlesen, welche Kommandozeilen-Parameter Sie übergeben müssen oder können, wenn Sie OTR Remote manuell starten, um eine Aufnahme hinzuzufügen oder zu löschen. Normalerweise werden Sie das niemals tun müssen, weil die verschiedenen Plugins die schmutzige Arbeit für Sie machen sollten. Die Kommandozeilen-Parameter werden detaillierter im Kommandozeilen-Referenz-Dokument beschrieben.
