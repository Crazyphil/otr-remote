Under the shiny surface

OTR Remote is controlled via the command-line interface. The Control Panel, etc. only let you comfortable set the surroundings, the main activity - adding and removing recordings from OTR - is controlled by command-line arguments.

Normally, you will never notice this fact, because the OTR Remote plugins do all the dirty work for you and configure the program correctly. But if there is no plugin which fits your needs, you will either have to start the program directly by using the command-line interface or develop your own plugin, which too needs knowledge about OTR Remote's internals.

In the following, we will show you which arguments are available and how to use them, as well as some examples.

Needed arguments: These arguments are always needed, when you want to add or remove a recording

-a

Indicates, that you want to add a recording job (either -a or -r)

-r

Indicates, that you want to remove a recording job (either -a or -r)

-h

Shows the Recording Help, which lets you read a short overview over this reference

-s="Station"

The short name of the station, from which you want to record your TV show

-sd=yyyy-mm-dd

The date corresponding to the TV show's start time in format <Year>-<Month>-<Day>

-st=hh:mm

The TV show's start time in format <Hour>:<Minute>

-et=hh:mm

The TV show's end time in format <Hour>:<Minute>, it may also be on the next day

Optional arguments: These arguments are optional, because they can be read from the settings or can be detected by OTR

-t="Title"

The title of the TV show you want to record

-u="User"

Your OTR username or E-Mail address

-p="Password"

Your OTR password in clear-text

-tz=Timezone

The timezone your times are input in (hours difference from UTC)

Examples:

Add the TV show "Mad TV" on station "Sunshine TV", which plays from November 19, 2010 from 20:00 to 23:30. We are located in Moscow (GMT+3)

OTRRemote.exe -a -s="Sunshine TV" -sd=2006-11-19 -st=20:00 -et=23:30 -t="Mad TV" -tz=3

Remove the news on station "CNN" on October 25, 2010 from 13:12 to 13:17. We want to use a different password as the saved one.

OTRemote.exe -r -s="CNN" -sd=2006-10-25 -st=13:12 -et=13:17 -p="secret"
