Programmoptionen

Konfigurieren, bis es passt

Die OTR Remote-Programmoptionen konfigurieren das allgemeine Verhalten der Anwendung. Der Dialog ist aufgeteilt in aufgabenbezogene Tabs zur einfacheren Navigation.

Auf dem "Programm"-Tab können Sie angeben, wie der Fortschritt des Hinzufügens oder Entfernens einer Aufnahme visuell angezeigt wird. Die Optionen sind:

Fenster zeigen

Ein Fortschrittsfenster mit Fortschrittsbalken wird angezeigt.

Systray-Icon zeigen

Ein Symbol wird in Ihrem System-Tray angezeigt, Sprechblasen informieren Sie über Fehlschlag oder Erfolg. Durch Klicken auf das Symbol wird das Fortschrittsfenster angezeigt.

Wenn Sie Windows 7 oder neuer als Betriebssystem verwenden, wird das Symbol nicht im System-Tray angezeigt, sondern in der Taskleiste. Keine Sprechblasen werden angezeigt, allerdings zeigen die entsprechenden Farben (Grün, Gelb, Rot) der in das Taskleisten-Symbol integrierten Fortschrittsleiste den Status.

Verstecken

Keine visuelle Fortschrittsanzeige wird angezeigt. Sie werden nur im Fehlerfall informiert.

Anzeige schließen nach: Wenn eine Fortschrittsanzeige gezeigt wird, können Sie wählen, ob und nach welcher Zeit sie automatisch versteckt werden soll.

Aufnahmevorschau vor dem Senden anzeigen: Wenn ausgewählt, wird vor dem Senden eines Aufname- oder Löschauftrags der Aufnahmevorschau-Dialog angezeigt. In diesem Dialog können Sie die Eigenschaften der Aufnahme bearbeiten und Serienaufnahmen starten.

Bereits laufende Sendungen aufnehmen: Wenn ausgewählt, ermöglicht es OTR Remote, Aufnahmeaufträge für bereits laufende Sendungen hinzuzufügen und sie vollständig aufzunehmen.

Folgende Sendung ebenfalls aufnehmen: Wenn ausgewählt, fügt OTR Remote automatisch 5 Minuten zur Endzeit Ihrer Sendung hinzu, um OTR zu zwingen, auch die nächste Sendung aufzunehmen. Wenn die Sendung länger als erwartet dauert, können Sie sie einfach aus diesen zwei Teilen zusammensetzen.

Aufnahme-Löschen automatisch neu versuchen: Wenn ausgewählt, versucht OTR Remote einmalig automatisch erneut, Ihre Aufnahme zu löschen, wenn OTR meldet, dass Sie sie nicht aufgenommen haben. Dies ist nötig, wenn Sie eine Aufnahme kurz nachdem Sie sie hinzugefügt haben wieder löschen möchten, weil OTR sie dann noch nicht in Ihren Account integriert hat.

Wenn Sie diese Einstellung nicht auswählen, werden Sie jedes Mal, wenn Sie eine Aufnahme löschen möchten und OTR meldet, dass Sie sie nicht aufgenommen haben, gefragt, ob Sie das Löschen erneut versuchen wollen.

Wöchentlich nach Aktualisierungen suchen: Wenn ausgewählt, wird Crazysoft AppUpdate jede Woche nach dem Schließen des Kontrollzentrums gestartet. Sie können eine Aktualisierung jederzeit erzwingen, indem Sie den "Aktualisieren..."-Link am Ende des Dialogs anklicken.

Senderkorrekturen: Öffnet den Senderkorrektur-Dialog, mit dem Sie Unterschiede zwischen den Sendernamen Ihres EPGs und denen von OTR verwalten können.

Sie benötigen Schreibrechte für die Datei stations.xml im Installationsverzeichnis, um Senderkorrekturen bearbeiten zu können. Ansonsten können Sie nur lesend auf die enthaltenen Einstellungen zugreifen.

Auf dem "Netzwerk"-Tab können Sie angeben, wie OTR Remote sich mit den OTR-Servern verbindet. Wenn Sie einen Proxy-Server benutzen möchten, wählen Sie, ob die Einstellungen von Internet Explorer benutzt werden sollen, oder Sie können Ihren eigenen Proxy nutzen. Sie können optional die Authentifizierung aktivieren, wenn die Standard-Anmeldedaten von Internet Explorer nicht funktionieren oder Ihr benutzerdefinierter Proxy einen Benutzernamen verlangt.

Auf dem "Über"-Tab finden Sie allgemeine Informationen über OTR Remote. Sie erhalten auch Informationen über die Version, die Sie gerade benutzen.
