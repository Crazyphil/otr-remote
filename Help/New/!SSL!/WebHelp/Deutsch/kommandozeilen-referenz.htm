Kommandozeilen-Referenz

Unter der glänzenden Oberfläche

OTR Remote wird über die Kommandozeile gesteuert. Kontrollzentrum, etc., geben Ihnen nur die Möglichkeit, die Rahmenbedingungen komfortabel zu setzen, die Hauptaktivität - Hinzufügen und Entfernen von Aufnahmen bei OTR - wird über Kommandozeilen-Parameter gesteuert.

Normalerweise werden Sie diese Tatsache nie bemerken, weil die OTR-Remote-Plugins die ganze dreckige Arbeit für Sie erledigen und das Programm richtig konfigurieren. Aber wenn es kein Plugin gibt, das Ihre Bedürfnisse deckt, werden Sie entweder das Programm direkt über die Kommandozeile starten müssen oder Ihr eigenes Plugin entwickeln, was ebenfalls Wissen über das OTR-Remote-Innenleben erfordert.

Im Folgenden werden wir Ihnen zeigen, welche Parameter verfügbar sind und wie man sie benutzt, außerdem ein paar Beispiele.

Erforderliche Parameter: Diese Parameter werden immer benötigt, wenn Sie eine Aufnahme hinzufügen oder entfernen wollen

-a

Gibt an, dass Sie eine Aufnahme hinzufügen wollen (entweder -a oder -r)

-r

Gibt an, dass Sie eine Aufnahme löschen wollen (entweder -a oder -r)

-h

Zeigt die Aufnahmehilfe, die Ihnen eine kurze Übersicht über diese Referenz bietet

-s="Sender"

Der Kurzname des Senders, dessen Sendung Sie aufnehmen wollen

-sd=yyyy-mm-dd

Das zur Startzeit der Sendung gehörige Datum im Format <Jahr>-<Monat>-<Tag>

-st=hh:mm

Die Startzeit der Sendung im Format <Stunde>:<Minute>

-et=hh:mm

Die Endzeit der Sendung im Format <Stunde>:<Minute>, darf auch am nächsten Tag sein

Optionale Parameter: Diese Parameter sind optional, weil Sie aus den Einstellungen gelesen oder von OTR ermittelt werden können

-t="Titel"

Der Titel der Sendung, die Sie aufnehmen wollen

-u="Benutzer"

Ihr OTR-Benutzername oder E-Mailadresse

-p="Passwort"

Ihr OTR-Passwort in Klartext

-tz=Zeitzone

Die Zeitzone, in der Ihre Zeiten eingegeben werden (Stunden Differenz von der UTC)

Beispiele:

Füge die Sendung "Mad TV" auf Sender "Sunshine TV" hinzu, die von 19. November 2010, 20:00 bis 23:30 Uhr läuft. Wir sind in Moskau (GMT+3).

OTRRemote.exe -a -s="Sunshine TV" -sd=2006-11-19 -st=20:00 -et=23:30 -tz=3

Lösche die Nachrichten auf Sender "CNN" am 25. Oktober 2010, 13:12 bis 13:17 Uhr läuft. Wir mächten ein anderes Passwort als das gespeicherte verwenden.

OTRemote.exe -r -s="CNN" -sd=2006-10-25 -st=13:12 -et=13:17 -p="geheim"
