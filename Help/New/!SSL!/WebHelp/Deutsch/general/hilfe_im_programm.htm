Hilfe im Programm

Wenn Sie keine großen Hilfedateien lesen wollen

Crazysoft OTR Remote hat eine eingebaute kontextsensitive Hilfe. Mit diesem Hilfesystem können Sie eine kurze Beschreibung über die Funktion einer Programmkomponente erhalten. Die Hilfe im Programm ist so einfach und klar wie möglich, wenn Sie aber mehr Details über eine Funktion wissen möchten, sollten Sie diese Hilfe-Dokumente verwenden.

Die Hilfe für eine Komponente wird angezeigt, wenn Sie eine kurze Zeit mit der Maus auf sie zeigen.
