When you don't want to read large help files

Crazysoft OTR Remote has a built-in context-sensitive help. With this help system, you can get a short description of what a program component does. The In-Program-Help is as simple and clear as possible, but if you want to know more details about a function, you should use this help documents.

The help for a component is shown when you point your mouse at it for a short period of time.
