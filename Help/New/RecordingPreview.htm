Before really adding or deleting a recording, and if you have enabled this setting in the Program Preferences, you can once again view and edit the recording's properties. The Recording Preview is also the right place for you to make series recordings of you favourite series.

Series Recordings

The Recording Preview dialog has an outstanding speciality for all fans of series. Here you can record a whole season of a series including the right episode titles in a totally easy way!

To activate the Series Recording function, enter the number of days following the day of the current recording. Those will be the base for calculating the following episodes. Also choose the interval in which you want to record the episodes. The intervals are:

Interval

Description

Recordings / Week

daily

A recording is added on each of the entered days.

7

on week day

Within the entered days, a recording is added on each equal week day as the original recording. (Example: Tuesday)

1

on workdays

Within the entered days, a recording is added on each workday, that is between Monday and Friday.

5

on weekends

Within the entered days, a recording is added on each weekend, that is Saturday and Sunday.

2

 For following recordings, the current recording's properties are copied 1:1, only the date is adjusted. After that, all recordings are added at once, that means, you have to start OTR Remote only for a single recording, configure the Series Recording and are able to add a whole season in one turn!

If you don't want to calculate the number of days, click onto the arrow up in the input field for the days and remain on it until on the bottom left the correct number of recordings is shown.

Organizing Series Recordings

If you want to organize your series recordings, one of the common sorting patterns is season and episode number. The recording title (and later also the filename) would be something like The Simpsons S05E21. To get all series recordings added by OTR Remote into such a scheme, you have to be able to increase a number in the title for every recording added. OTR Remote is able to increment numbers in the recording title (and only in the recording title) if you add a special marker around it. This are the { and } braces.

Applied to above example, you would add the recording title in OTR Remote by writing The Simpsons S05E{21}. This would mean, that the first recording has the number 21, and every following recording added in this step gets a number one higher than the previous. Of course, you can add as many auto-incrementing numbers as you want. Also, the number of digits is kept: Enter {0001} and all recordings will get a number with 4 digits.
