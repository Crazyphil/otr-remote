Methods: Enable

Are you there?

The Enable() method is called after all plugins are found and loaded. In this method, you can search for a specified program, check access rights, ... Depending on wether your requirements are met, return a boolean value and set the Enabled property to the same value.

Definition

bool Enable();

Sample

private bool _Enabled;
public bool Enabled
{
    get { return _Enabled; }
}

public bool Enable()
{
    try
    {
        FileInfo fi = new FileInfo(settings["Destination Folder"].Value + "\\" + settings["Destination Filename"].Value);
        FileStream fs = fi.OpenWrite();
        fs.Close();

        fi.Delete();
        _Enabled = true;
    }
    catch
    {
        _Enabled = false;
    }

    return _Enabled;
}
