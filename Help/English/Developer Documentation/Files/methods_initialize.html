Methods: Initialize

The warm-up

The Initialize() method is called, when the Control Panel searches for and loads plugins. This is done at start-up time of the program. In this method, you should do any work which is essential for later use of your plugin. Please keep this method short, so that the Control Panel can be loaded quickly.

Note: Do not execute any action not directly needed for initializing your plugin! You can do additional work (e.g. detecting if a specified program is installed) later in the Enable() method!

Definition

void Initialize();

Sample

private PluginSettingsCollection settings;

public void Initialize()
{
    string myDocuments = Environment.GetFolderPath(Environment.SpecialFolder.MyDocuments);

    settings = new PluginSettingsCollection();
    settings.Add("Destination Path", myDocuments, Crazysoft.OTR_Remote.ValueType.String);
    settings.Add("Destination Filename", "OTRRemote.bat", Crazysoft.OTR_Remote.ValueType.String);
}
