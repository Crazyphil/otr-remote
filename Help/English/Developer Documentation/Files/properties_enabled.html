Properties: Enabled

What do you want to do today?

The Enabled property is read-only and can be read anytime after the Enable() method was called. You should set this property according to the value, the method returns. The Enabled property indicates the Control Panel if the plugin can be used. More about this on the Enable() method's page.

Definition

bool Enabled { get; }

Sample

private bool _Enabled;
public bool Enabled
{
    get { return _Enabled; }
}

public bool Enable()
{
    try
    {
        FileInfo fi = new FileInfo(settings["Destination Folder"].Value + "\\" + settings["Destination Filename"].Value);
        FileStream fs = fi.OpenWrite();
        fs.Close();

        fi.Delete();
        _Enabled = true;
    }
    catch
    {
        _Enabled = false;
    }

    return _Enabled;
}
