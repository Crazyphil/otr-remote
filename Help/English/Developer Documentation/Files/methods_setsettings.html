Methods: SetSettings

Change your mind

The SetSettings() method is called, when the user has clicked the "OK" button and wants to save the work. The Control Panel passes a PluginSettingsCollection to this function, containing all settings you previously returned with the GetSettings() method.

Note: Only the settings you are returned here are available to you later when doing the save work. The Control Panel's settings (username, password, timezone) and the settings in the Preferences dialog are not visible to your plugin because of security reasons!

Definition

PluginSettingsCollection SetSettings();

Sample


private PluginSettingsCollection settings;

public void SetSettings(PluginSettingsCollection newSettings)
{
    settings = newSettings;
}
