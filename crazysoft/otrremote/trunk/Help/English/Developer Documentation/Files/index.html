Welcome to the PiKaSoft OTR Remote Developer Documentation!

TV guide plugins make OTR Remote fully dynamic and provide the user with various possibilities for connecting a TV guide program with OTR.com. You don't have to write a plugin explicitely for this purpose, but it would be the common usage.

The major thing to mind is, that plugins are only used in the Control Panel. They should normally create the possibility for the user to add a recording job without having to deal with commandline arguments. You could, for example, create a batch file, which asks the user to enter the data of a TV show and starts OTR Remote.

The following pages will show you how to use the OTR Remote API (OTRAPI.dll) to register a plugin in the Control Panel and work with the methods the API provides. Please note: We will reference to the sample plugin, which is only available when you have selected the developer package when installing.
