Program Preferences

Configure until it fits

The OTR Remote Program Preferences configure the general behaviour of the application. The dialog is splitted into task related tabs for easier navigation.

On the "Program" tab, you can specify, how the progress of adding or removing a recording is visually shown. The options are:

Show Window

A progress window with progress bar is shown.

Show Systray Icon

An icon is shown in your System Tray, balloon icons inform you of errors or success. By clicking on the icon, the progress window can be opened.

Hide

No visual progress indicator is shown. You are only informed in case of an error.

Hide indicator after: When showing a progress indicator, you can set if it should be hidden automatically and after which time.

Record shows, which already begun: When checked, OTR Remote enables you to add recording jobs for shows which are already running and record them completely.

Also record following show: When checked, OTR Remote automatically adds 5 minutes to the end time of your show, forcing OTR to also record the following show. If the show lasts longer than expected, you can easily reassemble it from those two parts.

Automatically retry recording deletion: When checked, OTR Remote automatically once retries deleting your recording when OTR reports that you don't have recorded it. This is necessary when you want to delete a recording a short time after adding it, because then OTR will not have integrated it into your account yet.

If you don't check this option, you will be asked to retry deletion every time you want to delete a recording and OTR reports that you don't have recorded it.

Automatically check for updates weekly: When checked, Crazysoft AppUpdate is started each week when closing the Control Panel. You can force an update anytime by clicking the "Check for updates..." link at the bottom of the dialog.

Edit Station Corrections: Opens the Station Corrections dialog to maintain differences between your EPG's TV station names and those of OTR.

You need Write access to the file stations.xml in the installation directory to be able to edit Station Corrections. Otherwise, you can access the contained settings read-only.

On the "Network" tab, you can specify how OTR Remote connects to the OTR servers. When you want to use a proxy server, you have to choose if the settings of Internet Explorer are used, or you can select your own proxy. You can optionally enable authentication, if the default credentials of Internet Explorer don't work or your custom proxy requires an username.

On the About tab, you can read general information about OTR Remote. You can also get information about the version you are currently using.
