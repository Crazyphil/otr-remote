Warten auf Fertigstellung

Der letzte Schritt des Hinzufügens oder Entfernens eines Aufnahmeauftrags wird durch den Aktionsfortschritt-Dialog (abhängig von Ihren Einstellungen) visualisiert. In dieser Phase werden Sie eventuell Firewall-Warnungen sehen, weil sich OTR Remote zu den OTR-Servern verbindet. Nachdem der Prozess abgeschlossen ist, können Sie das Fenster schließen oder warten, bis es automatisch geschlossen wird (falls aktiviert). Wenn ein Fehler auftritt, wird eine Nachricht angezeigt und Sie sollten das Problem beheben. Lösungen für häufige Probleme sind im Problembehandlungs-Dokument verfügbar.
