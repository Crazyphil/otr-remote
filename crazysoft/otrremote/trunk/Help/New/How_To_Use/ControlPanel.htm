Control Panel

Customize!

The OTR Remote Control Panel is the central place where you configure all settings, from your OTR login data to the network settings.

The most important thing to configure are your OTR username and your password. The password is saved DES-encrypted on your hard disk. When you don't enter it here, you will have to provide your data each time you add a recording via OTR Remote.

The next setting is the timezone. Each time you add a recording, start and end time have to be specified. OTR can calculate the correct times using the timezone you enter here. If you enter a wrong timezone, it may be possible that some other TV show than expected is recorded because of the incorrect times.

OTR Remote allows you to use plugins to configure TV guide programs to use OTR Remote for recording TV shows. The drop-down list lists all plugins currently available.

When you select a plugin, the "Plugin Settings" section changes. This section is freely configurable by the plugin, so you have to refer to the plugin's manual if you have any questions about this settings.

The "Preferences" button opens the Program Preferences dialog, in which you can configure program behaviours, network connection settings and update the program. The "Help" button opens this help document.

When you click the "OK" button, the settings are saved and the plugin is requested to do its work. When everything goes fine, the dialog closes. When you click the "Cancel" button, any changes (except those in the preferences dialog) are discarded.

If you have enabled the Auto-Update setting, Crazysoft AppUpdate is launched every week when closing the program to ensure your version is the latest one.
